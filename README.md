# Dasboard_PowerBI

**Criação de Insights e Storytelling com Dados para Apoio à Gestão**

***

### **Descrição**

Projeto tem como objetivo transformar dados brutos em informações estratégicas, utilizando técnicas de análise e visualização para gerar insights relevantes e construir storytelling orientado à tomada de decisão.

As principais entregas incluíram:

*   **Coleta e tratamento de dados** (ETL) com SQL, Power Query e Python.
*   **Criação de dashboards interativos** no Power BI para acompanhamento de KPIs.
*   **Análise exploratória e identificação de padrões** para otimizar processos internos.
*   **Storytelling com dados**: apresentação clara e visual para facilitar a compreensão por gestores.

**Impacto:** Redução do tempo de análise, maior agilidade na tomada de decisão e suporte à gestão estratégica.

***

### **Tecnologias utilizadas**

*   **SQL** para tratamento de dados.
*   **Power Query** para tratamento de dados.
*   **Python (pandas, matplotlib)** para análise e automação.
*   **Power BI** para visualização e storytelling.
